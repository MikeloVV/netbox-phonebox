CSV Format

Your CSV file should have the following columns:

  • number (required) - Phone number in international format
  • type (optional) - mobile, landline, voip, fax, emergency, toll_free
  • status (optional) - active, reserved, deprecated
  • country_code (optional) - RU, US, GB, etc.
  • provider (optional) - Provider name
  • contact (optional) - Contact name or ID
  • device (optional) - Device name or ID
  • description (optional) - Description
